4/*
5 * cxMultiLineInput.h - This class represents a multi-line input with a label.
6 * Logically, this is one big input, and the value will be spread across all
7 * lines. Wrapping is handled as the user types to the end of each line.
8 * This class is made up of single-line cxInput objects; however, in
9 * retrospect, this seems rather janky; it would be good to just eliminate
10 * cxInput, rename this class to cxInput, and re-do this class so that it is
11 * just a single window and can handle scrolling.
12 *
13 * Copyright (C) 2005-2007 Michael H. Kinney
14 *
15 * \author $Author: erico $
16 * \version $Revision: 1.155 $
17 * \date $Date: 2007/12/04 00:23:46 $
18 *
19 * Date User Description
20 * 08/23/07 erico Added a built-in onKeypress function to do per-keypress
21 * validation for numeric input types, and to allow deriving
22 * classes to override it to add additional functionality.
23 */
